# DataForge -- App Store Listings

## Apple App Store

**App Name:** DataForge -- AI Database Explorer
**Subtitle:** Ask your database questions in plain English
**Category:** Business / Productivity
**Price:** Free

**Description:**
DataForge turns your database into a conversation. Connect your PostgreSQL, MySQL, MongoDB, or SQLite database and ask questions in plain English. Get instant charts, dashboards, and insights -- no SQL required.

**Key Features:**
- Natural language queries -- ask questions, get instant answers with charts
- Upload CSV, JSON, Parquet files and explore them visually
- Data profiling -- automatic quality checks, outlier detection, PII warnings
- Transform pipelines -- clean and prepare data for ML training
- Export to CSV, Parquet, JSON, HuggingFace, or S3
- Beautiful auto-generated dashboards with 10+ chart types
- Team collaboration with role-based access control
- Scheduled reports via email, Slack, or webhooks
- Offline mode -- view cached dashboards without internet
- Voice input -- ask questions by speaking
- Secure -- AES-256 encryption, CSRF/SSRF protection, audit logging

**Keywords:** database, SQL, data, analytics, dashboard, charts, AI, machine learning, data prep, visualization

**What's New:**
Version 1.0 -- Initial release with full data pipeline: Upload -> Profile -> Transform -> Split -> Export
